## 🇩🇴 Sobre el Proyecto
Este trabajo forma parte del programa de becas Talento Digital. El objetivo es demostrar la integración de una interfaz de usuario moderna con validaciones robustas en el frontend y lógica estructurada en el backend.

## 🛠️ Especificaciones Técnicas
- **Frontend:** HTML5 Semántico y Tailwind CSS.
- **Validación de Formularios:** Restricción de caracteres numéricos en el campo de nombre mediante RegEx y eventos de entrada.
- **Backend:** Controlador en Java para la gestión de datos de contacto.
- **Control de Versiones:** Repositorio gestionado con Git y alojado en GitHub.

## 📁 Contenido
- `Index.html`: Interfaz principal.
- `ContactoController.java`: Lógica de procesamiento en Java.
- `Imagenes de proyecto/`: Recursos gráficos.
